# 손 안의 선생님 : 어플 안내

## 주제

디지털 약자 계층을 위한 스마트폰 안내 시스템

## 아이디어
휴대폰 가이드 : 디지털 약자 문제

- 예상 활용 기능 및 도구 : 안드로이드 스튜디오, AI 챗봇, LLM 모델, 앱 오버레이 기능
- 디지털 약자 계층이 주로 사용하는 플랫폼(유튜브, 구글, 네이버 등)을 지정해서 앱 실행 시 가이드 제공
- 각종 예약, 예매 서비스 가이드 제공
    - 스포츠 경기 예매, 고속버스/KTX 예매, 콜택시 등등
- 디지털 약자는 대다수가 노인이기 때문에 챗봇 연동한 말동무 서비스 부가 제공

### 차후 목표
AI 도우미에 카메라 기능을 도입하여 키오스크를 안내 해주는 등 디지털 약자에게 도움이 될 수 있는 부가 서비스 추가

### MVP
- **앱 가이드 시스템**
    - 디지털 약자 계층은 일반적인 앱 사용 방법이 미숙할 뿐만 아니라 이로 인하여 대중교통, 문화생활 이용에도 불이익을 겪고 있음. 이를 해결하고자 디지털 약자 계층에게 필요한 앱을 분석하여 앱 가이드를 제공함. 디지털 약자 계층도 손쉽게 예매, 예약, 또는 보편적으로 사용하는 앱의 사용 방법을 익힐 수 있도록 함.
- **자기효능감 달성**
    - 디지털 약자 계층이 타인에게 도움 받지 않고 필요한 기능을 혼자서도 사용할 수 있도록 도와줌. 주변인들에게 필요할 때마다 도움을 요청하는 것보다는 스스로 해볼 수 있는 기회를 제공하여 자기효능감을 느낄 수 있도록 함.
- **글자 대신 음성으로, 디지털 소외 계층의 정보 접근 지원**
    - 디지털 약자 계층은 소외 계층이 많으므로, 음성 가이드 서비스를 이용해 보다 친숙하고 쉽게 디지털 서비스를 이용할 수 있도록 함.
